Версия Telegram-бота с новой нейросетью.
Для корректной работы тг-бота требуется установить нужные библиотеки:
pip install pyTelegramBotAPI requests opencv-python numpy ultralytics onnxruntime

Также потребуется перенести файлы: best.onnx, InstructionV1.3.txt, SecurityCamsV2.1.py в один каталог.
